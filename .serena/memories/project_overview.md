# Project Overview

## Purpose
This is a basic Vite + React TypeScript application (vite-react-v1) that appears to be a starter template with the default Vite React setup.

## Tech Stack
- **Frontend Framework**: React 19.1.1
- **Build Tool**: Vite 7.1.2
- **Language**: TypeScript 5.8.3
- **Package Manager**: npm (based on package-lock.json)
- **Linter**: ESLint 9.33.0
- **Type Definitions**: @types/react, @types/react-dom

## Project Structure
```
/
├── public/           # Static assets
├── src/             # Source code
│   ├── assets/      # App assets (react.svg)
│   ├── App.tsx      # Main App component
│   ├── main.tsx     # Entry point
│   ├── App.css      # App styles
│   ├── index.css    # Global styles
│   └── vite-env.d.ts # Vite type declarations
├── package.json     # Dependencies and scripts
├── vite.config.ts   # Vite configuration
├── tsconfig.json    # TypeScript configuration
├── eslint.config.js # ESLint configuration
└── index.html       # HTML template
```

## Current State
- Basic Vite React template with a counter component
- Uses functional components with hooks (useState)
- Includes standard Vite + React logos and boilerplate content
